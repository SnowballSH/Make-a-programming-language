### In our language, we will have some syntax like:

- Var assignment:
```python
a = 3
b_0 = a + 1
```

- Function assignment:

```javascript
// regular function
f = function (arg1, arg2) {
    return arg1 - arg2 + 3
}

// lambda function (similar to real world algebra)
g(x, y) = 3x + y
```

- Call function:

```javascript
f(1, 2)
// or,
f 1, 2
```
